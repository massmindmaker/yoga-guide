1. Создание группы

Визард из 3 шагов. Откройте Группы → Создать.

Шаг 1 — Основное
Название и описание группы. Тип: «Регулярные» (еженедельные) или «Интенсив» (разовое). Минимум учеников для старта.
Шаг 2 — Цена и расписание
Ценообразование:
• Фиксированная — списывается 1 занятие с баланса
• Динамическая — цена назначается после голосования
• Наличные — ученик платит тренеру напрямую
Расписание: выберите дни недели и время.
Шаг 3 — Telegram-чат
Привяжите чат: создайте группу в Telegram → добавьте @Yom23_bot → сделайте админом → вставьте ссылку. Проверьте итог и нажмите «Создать».

3. Создание голосования

Визард из 3 шагов. Откройте группу → Создать голосование.

Требование: к группе привязан Telegram-чат
Шаг 1 — Настройки
Название голосования, период («На день» или «На неделю»), дедлайн.
Шаг 2 — Дни
Выберите дни из расписания группы. Для дневного — 1 день, для недельного — несколько.
Шаг 3 — Режим оплаты
• С баланса — списание занятия при голосовании
• Картой — оплата через T-Bank после голосования
• Наличными — оплата тренеру напрямую
• Бесплатно — простой опрос
Результат: опрос автоматически публикуется в Telegram-чат

4. Управление голосованиями

📨 Отправить в чат: публикует голосование как нативный Telegram Poll. Под опросом — кнопки оплаты: картой, с баланса или наличными
✏️ Редактировать: диалог для изменения названия, дедлайна и минимума участников
✅ Финализировать: доступно только при набранном кворуме. Для динамической цены — тренер назначает стоимость

5. Дашборд, журнал и аналитика

1. Главная и расписание

Главная: приветствие с аватаром из Telegram, текущий баланс, ближайшие занятия с датой, группой и статусом
Гостевой режим: если нет Telegram — можно ввести имя вручную (сохраняется локально)
Расписание: полоска дней недели. Для каждого дня — список занятий с группой, тренером, свободными местами

Индикаторы: оранжевый «Осталось N» при ≤3 местах, серый «Мест нет» при 0, зелёный «Вы записаны» при бронировании

2. Голосование

Голосования приходят в Telegram-чат как нативный опрос + кнопки оплаты
Можно голосовать прямо в чате или через Mini App
Карточка голосования: акцентная полоска (мятная = активно), прогресс кворума, мини-аватары голосовавших
Выберите варианты (подсвечиваются мятным), нажмите Проголосовать
Если включено списание — кнопка показывает «Проголосовать (1 занятие)». При недостатке баланса — ошибка с текущим/нужным количеством
После голосования: зелёная метка «Вы проголосовали» + кнопки «Оплатить» и «Списать с баланса»
При отмене голосования тренером — баланс возвращается автоматически
